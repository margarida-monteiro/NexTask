# NexTask

NexTask é uma aplicação web, desenvolvida em Django do estilo Trello, criada para organizar tarefas de forma simples e intuitiva através de boards e listas.

A aplicação permite visualizar diferentes boards, gerir tarefas e acompanhar o estado de cada uma (To Do, Doing e Done).

Este projeto foi desenvolvido no âmbito do módulo de Administração de Sistemas com o objetivo de aplicar conceitos como:

- Python e Django  
- Git e GitHub  
- Docker e Docker Compose  
- Base de dados MySQL
